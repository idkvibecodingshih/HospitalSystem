# 🏥 HospitalSystem

A simple yet practical hospital data management system built with Python.

This project allows users to input patient information, organize structured records, and export the data into a CSV file that can be easily opened and managed in Excel.

---

## 🚀 Features

- Patient registration via terminal input
- Structured data handling using dictionaries/lists
- CSV export compatible with Excel
- Clean and scalable code structure
- Ready for future expansion (database, API, UI)

---

## 🧠 How It Works

1. The user enters patient data (name, age, diagnosis, etc.)
2. The system stores the data in structured format
3. Data is exported into a `.csv` file
4. The file can be opened and managed in Excel
